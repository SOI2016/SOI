To Compile the APIs and to create the shared library
----------------------------------------------------
administrator@administrator-HCL-Desktop:Working$ gcc -c -Wall -Werror -fpic shm_que.c
administrator@administrator-HCL-Desktop:Working$ gcc -shared -o libshm.so shm_que.o
administrator@administrator-HCL-Desktop:Working$ sudo cp libshm.so /usr/lib/
administrator@administrator-HCL-Desktop:Working$ sudo chmod 0755 /usr/lib/libshm.so
